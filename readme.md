# Agro Forte Sustentável

## Tema
Produção e Meio Ambiente

## Objetivo
Promover a conscientização sobre práticas sustentáveis no agronegócio, demonstrando a importância da preservação ambiental juntamente com a produção agrícola.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Informações sobre sustentabilidade no campo
- Layout responsivo
- Contador interativo de árvores preservadas
- Botão informativo sobre agricultura sustentável
